# Project Plans

This file is the index for implementation plans that live under `projects/`.

Purpose:

- keep project-specific plans out of published site pages
- give humans and AI agents one predictable place to look first
- point to the current working plan for each project

Current plans:

- `castle-ledger`: [projects/castle-ledger/PLAN.md](/home/arominge/repos/AlanCoding.github.io/projects/castle-ledger/PLAN.md)

Conventions:

- put a `PLAN.md` inside a project folder when that project has active or near-term work
- keep it short, concrete, and append-only enough that future agents can recover context quickly
- if a plan becomes obsolete, update the project entry here rather than deleting the trail entirely
